# WebsiteMaker - Complete Feature List

## Multi-Page Website Support

**Create complete multi-page websites!**
- Add unlimited pages to your website
- Each page can have its own unique sections
- Automatic navigation menu linking all pages
- Download as a ZIP file containing all HTML files
- Single-page websites download as a single HTML file

### How to Use Multi-Page:
1. Click "+ Add Page" in the Pages section
2. Name your page (e.g., "About Us", "Services")
3. Set the filename (e.g., "about.html", "services.html")
4. Click "Edit Sections" to add content to that page
5. Switch between pages to edit different content
6. Download creates a ZIP with all pages linked together

## Design & Layout Options

### Color Themes
- **Custom** - Choose your own colors
- **Professional** - Blue & Gray theme
- **Vibrant** - Purple & Orange theme
- **Pastel** - Soft Pink & Blue theme
- **Dark Mode** - Dark background with Cyan accents
- **Nature** - Green & Brown theme
- **Sunset** - Orange & Pink theme

### Typography
- **Heading Font** - Separate font for headings (6 options)
- **Body Font** - Font for body text (6 options)
- **Font Size** - Small, Medium, or Large
- **Line Height** - Compact, Comfortable, or Spacious

### Layout Controls
- **Page Width** - Narrow (800px), Standard (1200px), Wide (1400px), or Full Width
- **Spacing** - Compact, Comfortable, or Spacious section spacing
- **Border Radius** - Sharp, Slightly Rounded, or Very Rounded corners
- **Shadow Intensity** - None, Subtle, Medium, or Strong shadows

### Button Styling
- **Button Style** - Filled, Outline, or Ghost
- **Button Size** - Small, Medium, or Large

### Other Options
- **Animations** - Enable or disable transitions
- **Content Alignment** - Left, Center, or Justified text

## Header & Footer

### Header Options
- Toggle header on/off
- **Logo Type** - Text or Image URL
- **Navigation Items** - Comma-separated menu items
- **Header Style** - Sticky (stays on top) or Static

### Footer Options
- Toggle footer on/off
- **Copyright Text** - Custom copyright message
- **Social Links** - Format: Platform|URL (comma-separated)

## Accessibility Features

All sections include:
- Proper ARIA labels and roles
- Keyboard navigation support
- Screen reader compatibility
- Focus indicators
- Skip links
- Semantic HTML structure

## Usage

1. Fill in basic website information (title, heading, content)
2. Choose a color theme or customize colors
3. Configure typography and layout options
4. Add header/footer if desired
5. Add sections using the "+ Add Section" button
6. Configure each section with its specific options
7. Preview your website
8. Download the HTML file

Your generated website will be a single, self-contained HTML file with inline CSS!
